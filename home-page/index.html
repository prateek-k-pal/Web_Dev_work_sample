About the Website

This page was dessigned as a project for different puposes and the creator is Prateek Kumar Pal. It is made using HTML as the for the primary layout. CSS for the styling and effect with SCSS as the preprocessor. JavaScript was used for algorithms and related functions. The webpages and the project itself was made with CodePen . To learn more about the creator go to the 'About the Creator' page. You are invited to take a tour of the website and leave a feedback on the 'Contact Us' page.


Functions of the Website

  • Home: This button on the dashboard brings you back to the home page of the website.

  • Image Filters: These filters are made with JavaScript and can be used to edit your images. To use the filter function just click on the 'Choose File' button and after selecting the image you will see it getting uploaded you can then choose any filter of your choice to edit the image. You can also download the edited pic right-clicking on the Output pic and saving the image or using a similar method in mobiles.

  • Paint Canvas: It is a basic painting tool which can be used to write or create anything else you like. Using it is very simple, you can choose the size of the brush, the dimensions of the canvas, the color of you pen and you are ready to draw. To reset the drawing area just click on the 'Clear' button. The functions the webpage are made with JavaScript.

  • Steganography: It is defined as the art of hiding secret data in a non-secret or normal file. With this page you can perform basic steganography of hiding an picture in another. The only thing you have to consider while using it is that both the pictures should have similar colour tones.

  • About the Creator: This page provides you with an introduction to the creator and ways to contact him directly.

  • Contact Us: The page consists of a form which can be filled to provide the creator a feedback on the website. You can also get in touch with the creator this way.
